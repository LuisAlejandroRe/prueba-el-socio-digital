## About The Project

El router permite entrar a categoria por param (e.g. https://kuksfresh-demo.web.app/category/Lamb) y a receta por si id (e.g. https://kuksfresh-demo.web.app/meal/53009)

## Installation

instalar.

```sh
npm i
```

Iniciar servidor de desarrollo:

```sh
npm start
```
